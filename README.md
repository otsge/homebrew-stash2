# Otsge Stash2

## How do I install these formulae?

`brew install otsge/stash2/<formula>`

Or `brew tap otsge/stash2` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "otsge/stash2"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
